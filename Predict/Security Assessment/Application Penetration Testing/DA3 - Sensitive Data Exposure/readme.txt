
Unintentional exposure of sensitive information such as PII, financial information, 
healthcare information or application keys and secrets. This may include data in 
memory post app logout, logs with sensitive info., hardcoded secrets in files (dll, 
binaries, configuration files), etc. An attacker can use this information to carry 
out identity frauds.
