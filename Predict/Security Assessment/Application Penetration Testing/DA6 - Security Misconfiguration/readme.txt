
Flaws include misconfigured group policies / registry / firewall rules etc., 
missing file-content type check for file processing apps, misconfigured 
named-pipes, misconfigured supporting services used by the application, 
misconfigured third party services (SQL, AD, etc). An attacker can gain system 
access by exploiting these flaws.
