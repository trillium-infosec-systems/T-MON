
ID	T0871
Tactic	Execution
Data Sources	API monitoring, Network protocol analysis, Packet capture
Asset	Field Controller/RTU/PLC/IED

Adversaries may attempt to leverage Application Program Interfaces (APIs) 
used for communication between control software and the hardware. Specific 
functionality is often coded into APIs which can be called by software to 
engage specific functions on a device or other software.
