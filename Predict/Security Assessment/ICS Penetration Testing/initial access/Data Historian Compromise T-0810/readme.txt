

ID:		T0810
Tactic:		Initial Access
Asset:		Data Historian


Adversaries may compromise and gain control of a data historian to gain a foothold into 
the control system environment. Access to a data historian may be used to learn stored 
database archival and analysis information on the control system. A dual-homed data historian 
may provide adversaries an interface from the IT environment to the OT environment.
