ID:		T0818
Tactic:		Initial Access
Data Sources:	File monitoring, API monitoring, Windows event logs
		External Contributors	Joe Slowik - Dragos
Asset:		Engineering Workstation

Adversaries may compromise and gain control of an engineering workstation 
for Initial Access into the control system environment. Access to an 
engineering workstation may occur through or physical means, such as a 
Valid Accounts with privileged access or infection by removable media. 
A dual-homed engineering workstation may allow the adversary access into 
multiple networks. For example, unsegregated process control, safety system, 
or information system networks.
