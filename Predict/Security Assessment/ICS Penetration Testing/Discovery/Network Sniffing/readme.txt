	Network Sniffing
Technique ID:	T0842
Tactic:		Discovery
Asset:		Field Controller/RTU/PLC/IED

Network sniffing is the practice of using a network interface on a computer system to 
monitor or capture information. regardless of whether it is the specified destination 
for the information.
