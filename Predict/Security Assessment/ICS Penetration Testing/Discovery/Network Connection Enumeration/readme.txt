	Network Connection Enumeration
Technique ID:	T0840
Tactic:		Discovery
Asset:		Human-Machine Interface

Adversaries may perform network connection enumeration to discover information about 
device communication patterns. If an adversary can inspect the state of a network 
connection with tools, such as netstat, in conjunction with System Firmware, then 
they can determine the role of certain devices on the network
