
ID:		T0881
Tactic:		Inhibit Response Function
Data Sources:	Process command-line parameters, Process monitoring, Windows Registry, API monitoring
Asset:		Human-Machine Interface, Control Server, Data Historian, Engineering Workstation

Adversaries may stop or disable services on a system to render those services unavailable 
to legitimate users. Stopping critical services can inhibit or stop response to an incident
or aid in the adversary's overall objectives to cause damage to the environment. Services 
may not allow for modification of their data stores while running. Adversaries may stop
services in order to conduct Data Destruction
