
ID:		T0835
Tactic:		Inhibit Response Function
Data Sources:	Controller program, Process monitoring
Asset:		Field Controller/RTU/PLC/IED

Adversaries may manipulate the I/O image of PLCs through various means to prevent them from 
functioning as expected. Methods of I/O image manipulation may include overriding the I/O 
table via direct memory manipulation or using the override function used for testing PLC programs.
