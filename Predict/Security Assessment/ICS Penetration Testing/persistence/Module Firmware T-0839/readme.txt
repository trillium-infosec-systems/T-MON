
ID:		T0839
Tactic:		Persistance
Data Sources:	Sequential event recorder, Digital signatures, Network protocol analysis, 
		Packet capture.
Asset:		Field Controller/RTU/PLC/IED, Safety Instrumented System/Protection Relay

Adversaries may install malicious or vulnerable firmware onto modular 
hardware devices. Control system devices often contain modular hardware
devices. These devices may have their own set of firmware that is separate
from the firmware of the main control system equipment.
