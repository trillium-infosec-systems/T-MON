	Directory traversal
Old computer routers can have serious flaws that enable remote adversaries 
to take control of them. A Kyle Lovett security researcher, for instance, 
found that more than 700,000 ADSL routers distributed to various customers 
suffered from the “directory traversal” vulnerability that provides hackers 
with a way to extract administrative details.
