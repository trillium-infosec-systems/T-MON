		Speculative Store Bypass (SBS)
A variant of the Spectre security vulnerability, SSB or Speculative Store 
Bypass enables hackers to execute memory readers before memory write 
addresses are revealed. It can also be used to leak cross-process data. 
The vulnerability impacts Intel, AMD and ARM variants of processors.
