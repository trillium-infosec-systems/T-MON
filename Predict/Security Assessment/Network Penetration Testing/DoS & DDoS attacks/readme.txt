	 DoS and DDoS Attacks
Website crashes also happen due to cyberattacks in the form of DoS and
DDoS attacks. As a result, the system crashes because of malicious 
traffic overload, and the users cannot access the website. These types 
of network security attacks aim to cripple the IT infrastructure of 
the victim network.
