		Two-Factor Authentication Interception

ID: T1111
Sub-techniques:  No sub-techniques
Tactic: Credential Access
Platforms: Linux, Windows, macOS

System Requirements: 	Smart card Proxy: Use of smart cards for single or multifactor authentication to access 
 to network resources. Attached smart card reader with card inserted; Out-of-band one-time code: Access to the 
 device, service, or communications to intercept the one-time code; Hardware token: Access to the seed and 
 algorithm of generating one-time codes.

Permissions Required: Administrator, SYSTEM
