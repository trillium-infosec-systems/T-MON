This threat refers to the misuse of any platform feature of the Android or iOS operating system or
failure to incorporate platform security controls. This includes issues concerning improper use of
security controls and platform features that are a part of the mobile operating system, such as:
1.Misuse of the iOS Touch ID feature, which can result in unauthorized access to the device.
2.Incorrect use of the iOS Keychain for instance by storing session keys in the app local storage.
3.Requesting excessive or wrong platform permissions
4.Android intents (used to request an action from another app component) that are marked public may
  reveal sensitive information or permit unauthorized execution.
